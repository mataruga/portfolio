Cohort Analysis Tool

This project was made in collaboration with KI (Karolinska Institutet) and Janssen J&J (supervised by Mario Romero at KTH) where we set out to visualize their (KI's) database over prostate cancer patients in Sweden. The project was meant to help them better understand their demographic by giving them the tools to create cohorts and find their own correlations and relationships. You can try out the tool here.

- Contributors: Danilo Mataruga, Filip Gustafsson, Julia Creutz, Raghu Nayyar, Nilay Ates, Mai-Khanh Lê, Niek Bijman, Tomas Albrecht.

NLUI For Chess

A NLUI is a "Natural Language User Interface", i.e. the user communicates with the program using natural language. This project consists of a NLUI implemented on chess. You tell the engine what move to make in natural language and it will try to figure out the optimal move, e.g. "capture black queen". It's implemented using python and can be downloaded here. A thorough report of the project can be downloaded here.

- Contributors: Danilo Mataruga.

Perception of Virtual Facial Expressions

This project is a modern interpretation of Palermo et al.:s experiment measuring the perception of virtual facial expressions. It was implemented in Unity using C#. A full report of the experimental design and results can be downloaded and read here.

- Contributors: Danilo Mataruga, Nilay Ates, Mai-Khanh Lê.

Umbvrella - VR game

The game utilizes a physical umbrella as a novel input method. The player controls their descent by tilting the umbrella, collecting points and avoiding bombs on their way to the landing zone. The game in made in Unreal Engine and coded in C++. More details about the game can be found here, and the github to the project here.

- Contributors: Danilo Mataruga, Nilay Ates, Mai-Khanh Lê.
